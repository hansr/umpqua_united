# Apex FC Bend — Site Spec

**Base URL:** https://apexfcbend.com/  
**Tagline:** The Premier Soccer Club in Central Oregon

---

## Site map

- **Home** `/`
- **All Programs** `/all-programs` — Browse/register for programs (Sprocket Sports)
- **External / off-site**
  - Tryout registration: `https://apexfc.sprocketsports.com/club/tryouts2026`
  - Contact / mailing list: `https://apexfc.sprocketsports.com/club/contact`
  - Development Academy info: `https://apexfc.sprocketsports.com/club/developmentacademy`
  - Open programs (register): `https://apexfc.sprocketsports.com/open-programs?programID=...`
  - Club merch: `https://www.boutitmerch.com/collections/apex-futbol-club`
  - Member dashboard: `https://apexfc.sprocketsports.com/dashboard` (Log in)

## Per-page information sections

### Home (`/`)

| Section | Description |
|--------|-------------|
| **Hero / CTA** | Rotating or repeated CTA: “Tryout registration now open” with link to register; “Apex FC – The Premier Soccer Club in Central Oregon” with “Join our mailing list” link. |
| **Welcome** | “Welcome to our club” — copy on joy, family, competitiveness, accountability, and developing better people through soccer. “Learn more” link. |
| **Partners** | “Our partners” — sponsor/partner area. |
| **Open programs** | List of current programs with **Info** and **Register** links: e.g. Development Academy (DOB 2016–2018), Youth Premier Academy tryouts, HS Premier Academy tryouts. “Browse more programs” → `/all-programs`. |
| **Club news** | “Club news” — e.g. newsletter (e.g. February 2026), “Download the Sprocket Sports App.” |
| **Sponsors** | Tiered: Pinnacle, Gold, Founding, Event sponsors; “Partners.” |

### All Programs (`/all-programs`)

| Section | Description |
|--------|-------------|
| **Find a program** | Heading + filters/browse. |
| **Program list** | Categories: “Core Program” (1 program), “Register for Team Tryouts” (2 programs). Links to Sprocket Sports for details/registration. |
| **Header** | Club logo (→ home), “Log in” (→ Sprocket dashboard). |

---

## Notes

- Registration, contact, and program details live on **Sprocket Sports**; apexfcbend.com is the public face and funnel.
- No deep in-site nav in fetched content; primary paths are Home and All Programs plus external CTAs.
